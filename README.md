# Forest Whisper

A sleep music player system built with ArkTS, designed for sleep and focus scenarios.

## Project Introduction
Forest Whisper is a team project focusing on sleep-aid and concentration scenarios.  
The system provides music playback, timed shutdown, gradual fade-out, content browsing, favorites management, and community interaction features.

## My Role
As a team member, I participated in:
- requirement analysis and system design
- player module development
- sleep timer and fade-out logic implementation
- content organization and part of interaction module development
- testing and module integration

## Main Features
- music playback and pause
- track switching and progress control
- background playback
- sleep timer
- gradual volume fade-out before stopping
- content browsing and search
- favorites and recent play records
- dynamic/community module

## Tech Stack
- ArkTS
- HarmonyOS
- AVPlayer
- RDB
- event bus
- cache service
